# メモアプリ - 要件定義書

# 1 概要
シンプルなメモアプリの作成
ユーザーが投稿を行い、一覧で閲覧できる最小限の機能を持つ。
メモの公開/非公開の選択を可能にする

# 2 目的
Webアプリ開発の基礎を学ぶため、Laravelを使用してメモアプリを作成する。
投稿機能や一覧表示など、Webアプリの基本的な処理の流れを理解・実装することを目的とする。

# 3　主な機能

# 3-1 アカウント登録/ログイン
- ユーザー登録/ログイン(メール認証)
- パスワードリセット
- laravel Fortify(Laravel12ではbreezeが非推奨のため)

# 3-2 メモの登録
- 登録ページからメモの登録を可能にする
- タイトル、サブタイトル、本文の登録

# 3-3 メモ一覧表示
- 自分が登録したメモを一覧表示
- 一覧ではタイトルとサブタイトルのみ表示

# 3-4 メモの詳細
- タイトル、サブタイトル、本文の表示
- 編集ボタンを設置
- 削除ボタンの設置

# 3-5 メモの編集
- モーダルで編集（登録と兼用）
- モーダルで削除確認

# 4 今後の拡張候補
- トップページでユーザーが登録したメモの表示(ユーザーのメモ共有機能  公開で登録した場合のみ)
- 各メモにコメントの投稿を可能にする
- マークダウン形式に対応したい
